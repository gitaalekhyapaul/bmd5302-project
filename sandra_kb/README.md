# Sandra Knowledge Base

This folder is the repo-local knowledge base for Sandra's chat layer.

Sandra uses these files as her advisory context:

- `sandra_preprompt.md`: runtime system prompt and behavioral guardrails
- `methodology.md`: report and workbook methodology distilled from the project report and `Model.xlsm`
- `tone_guide.md`: wording rules for warm, practical, methodical responses

The workbook remains the source of truth for live calculations. The knowledge base helps Sandra explain methodology, investment strategy, assumptions, and limitations; it must not replace workbook-generated outputs.
